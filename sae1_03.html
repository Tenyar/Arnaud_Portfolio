SAE1.03
Installation d'un poste de travail pour le développement

Préparation / Présentation globale

Ce projet avait pour but d'installer un système d'exploitation Linux (Debian 11).
Puis de faire une installation d'outils de développement dont un IDE (Intellij IDEA).

Cette S.A.É. est regroupé en deux thèmes :
- Installation d’un système d’exploitation (Debian) avec une interface graphique Plasma/KDE.
- Installation d’outils de développement (IDE, etc...).

C'était un projet individuel en autonomie.

Pour l'installation du système d'exploitation (OS) il se fera sur :
- Une machine virtuelle (MV) Sotckée sur une station linux.
- L'installation est sur une interface graphique.
- Nous installerons les packages nécessaires avec une distribution "apt".

Exemple machine virtuelle (avec Debian)

Ressources qui alimentent ce projet

Les ressources utilisées pour ce projet sont :
• R1.04 [Informatique : systèmes d’exploitation]
Elle comprend mes connaissances sur le système Debian.

• R1.11 [Base de la communication]
Elle comprend nos méthodes de communication lors d'un contexte projet.

Mon rôle

Le travail étant individuel j'ai réalisé ce projet en autonomie et individuellement.

Mes compétences acquises

À la fin de ce projet, j'ai acquis différentes compétences :

• Prise en main de l'installateur Debian.

• Installer un système d'exploitation / un système Linux.

• Configurer un système d'exploitation et le rendre adpaté à un besoin.

• Construire un bilan dans une langue étrangère sur notre travail.
