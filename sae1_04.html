Préparation / Présentation globale

Ce projet avait pour but d'installer un système d'exploitation Linux (Debian 11).
Puis de faire une installation d'outils de développement dont un IDE (Intellij IDEA).

Cette S.A.É. est regroupé en deux thèmes :
- Installation d’un système d’exploitation (Debian) avec une interface graphique Plasma/KDE.
- Installation d’outils de développement (IDE, etc...).

C'était un projet individuel en autonomie.

Pour l'installation du système d'exploitation (OS) il se fera sur :
- Une machine virtuelle (MV) Sotckée sur une station linux.
- L'installation est sur une interface graphique.
- Nous installerons les packages nécessaires avec une distribution "apt".

Exemple machine virtuelle (avec Debian)

Afin de limiter notre impact écologique, nous avons dû trouver une solution optimisée.

C'est-à-dire programmer un tri optimisé ainsi qu'un algorithme efficace.
L'optimisation se fait d'une part dans le temps d'exécution que met le programme, soit en millisecondes.

Mais aussi dans l'élaboration de l'algorithme et dans la qualité du développement.
Si notre programme était lisible et correctement commenté.
