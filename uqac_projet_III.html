Introduction et mise en contexte

Ce projet a pour but la conception, la réalisation et l'implémentation de l'outil de monitoring réseau Suricata. Tout ceci dans un espace docker (docker-compose) déployable rapidement. L'IHM est une interface web.

Retrouvez le dépôt GitHub du projet ici

Outils/ langages utilisées

Le language utilisé pour le développement de l'application est Python en majorité avec diverses librairies trouvables dans le requirements.txt.

Mes compétences acquises

À la fin de ce projet, j'ai acquis différentes compétences :

• Développer mes compétences vis à vis du monitoring réseaux avec Suricata.

• Exploitation d'un docker-compose avec sous-réseaux.

• Implémentation de journalisation, configuration en directe et statistiques du réseaux via une interface web.
