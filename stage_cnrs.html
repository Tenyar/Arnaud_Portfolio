Introduction et mise en contexte

J’ai effectué mon stage du 15 avril au 21 juin 2024 au sein de Mathdoc, une unité mixte du CNRS et de l’UGA, spécialisée dans les services documentaires et l’édition scientifique pour les mathématiques.
Mon stage s’est inscrit dans le cadre du développement et de l’automatisation de tests fonctionnels ainsi que de la création de nouvelles fonctionnalités pour le logiciel Open Journal Systems (OJS), utilisé par Mathdoc pour la gestion de revues scientifiques.

Retrouvez plus d'informations sur le rapport du stage ici

Outils/ langages utilisées

J’ai utilisé plusieurs langages et outils au cours du stage :
• JavaScript (notamment avec le framework Cypress pour les tests fonctionnels).
• PHP avec le framework Laravel pour le développement des plugins.
• Bash pour les scripts de déploiement de conteneurs Docker.
• Vue.js pour les interfaces utilisateurs.
• Smarty pour les templates HTML côté serveur.
J’ai également travaillé avec GitLab et ses runners CI/CD pour l’intégration continue.

Réalisation

J’ai migré d’anciens tests fonctionnels Selenium vers Cypress.
J'ai aussi développé deux plugins pour OJS (l’un ajoutant les emails à côté des noms des juges, l’autre restreignant les droits d’un utilisateur en lecture seule).
Conçu un script Bash permettant de déployer automatiquement un environnement de travail OJS dans un conteneur Docker.
J’ai aussi participé à des réunions hebdomadaires de coordination et suivi l’évolution de mes tâches via un tableau Trello et un carnet de bord.

Mes compétences acquises

Ce stage m’a permis d’approfondir mes compétences :

• PHP, JavaScript et Bash

• Utilisation des frameworks Laravel et Vue.js

• Tests automatisés avec Cypress

• Gestion de projet, travail en équipe, communication professionnelle

• Résolution autonome de problèmes complexes et adaptation rapide
