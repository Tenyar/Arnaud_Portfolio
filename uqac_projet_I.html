Introduction et mise en contexte

Ce projet a pour but la conception et la mise en œuvre d’un gestionnaire de mot de passe sécurisé permettant à un utilisateur de créer des comptes et de stocker des mots de passes pour chaque comptes.

Retrouvez le dépôt GitHub du projet ici

Outils/ langages utilisées

Le language utilisé pour le développement de l'application est Python, avec multiple librairies :
• cryptography (chiffrement)
• argon2-cffi (hashage)
• customtkinter (librairie graphique, vues)
• Pillow (librairie graphique)
• pyperclip

Ainsi que Mysql pour la base de données.
